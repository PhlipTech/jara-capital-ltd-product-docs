Backend – Logical entity-relationship diagram

Investment App Nigeria · App database (not Cowrywise internal schema) · Open in a browser. Markdown source in repo: ../investment-app-nigeria-erd.md (Mermaid in editor).

Principles: Cowrywise is the source of truth for balances, positions, and regulated transactions. This schema stores app identity, PhlipVerify eKYC outcomes, Cowrywise linkage IDs, consent, idempotency keys, sessions, admin audit, and optional sync cursors. COWRYWISE_LINKAGE.kyc_status reflects Cowrywise account state if their API exposes it; customer identity checks are not from Cowrywise. Optional MVP+ tables: CACHED_TRANSACTION, CACHED_POSITION for TTL-based read models.
